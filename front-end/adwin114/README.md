## TypeScript
Vanilla JS 랑 다르게 타입캐스팅 말고도 다양한게 가능하다 하고
파이썬에 익숙한 나에게 좋아보여서 사용하게 됐다.
CRA 에서 템플릿을 제공하기 때문에 삽질안해도된다.
